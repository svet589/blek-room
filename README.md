# _🖤 BLEK.ROOM 🔥_

###  *"Никаких серверов. Никакой х¥йни. Только ты и твой собеседник."*

BLEK.ROOM — это форк **Delta Chat**, который мы перешаманили под себя. Вся переписка идёт через децентрализованную сеть (email), а в будущем — через Tor и SOCKS5-прокси. Ни один ёблюдок тебя не прочитает. Даже если очень захочет.
